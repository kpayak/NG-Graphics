CTP

Computer-to-plate (CTP) is an imaging technology used in modern printing processes. In this technology, an image created in a Desktop Publishing (DTP) application is output directly to a printing plate. This compares with the older technology, computer-to-film (CTF), where the computer file is output onto a photographic film.
